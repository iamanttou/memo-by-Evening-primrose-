# 📒 memo-by-Evening-primrose-

シンプルでパステル調のブラウザメモアプリです。  
フォルダで整理しながら、メモをすばやく作成・編集できます。

---

## ✨ 特徴

- 📁 フォルダ管理
- 📝 メモ作成・編集
- 💾 自動保存（localStorage）
- 📌 ピン留め機能
- ⭐ お気に入り
- 🗑 ゴミ箱機能
- 🔍 検索機能
- ⌨ ショートカット対応（Ctrl+S / Ctrl+N）

---

## 🧠 使い方

1. `index.html` を開く
2. メモを作成する
3. 左で編集、右で管理
4. 自動で保存されます

※ サーバー不要、ブラウザだけで動きます
